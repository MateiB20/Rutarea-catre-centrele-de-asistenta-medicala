# Rutarea-catre-centrele-de-asistenta-medicala

  Lucrarea își propune să prezinte posibilitatea utilizării unei platforme capabile să identifice în timp real traseul optim între locația incidentului și unitățile medicale disponibile, luând în calcul nu doar distanța fizică, ci și dinamica traficului urban și gradul de ocupare al unităților medicale. 
  Aplicația îmbină o implementare originală a algoritmului de căutare A*, adaptată pentru a răspunde condițiilor de criză folosind, spre deosebire de soluțiile standard de navigație, un model de graf dinamic construit prin extragerea și fuzionarea mai multor rute alternative oferite de serviciul Mapbox API. 
  Rezultatele experimentale au indicat durate de traseu aproape identice pentru algoritmul A* și Best-First Search. Din perspectiva latenței, Best-First Search a obținut valori mai reduse, însă A* oferă o evaluare mai riguroasă prin luarea în calcul a costului total al traseului.

Cuvinte cheie: Sisteme informatice geografice (GIS), Algoritmul A* optimizat, Managementul transportului de urgență, Grafuri dinamice, Optimizarea traficului urban.
